`rs-grid-yew` provides a `GridCanvas` function component for Yew CSR
applications. It wraps the WASM runtime, canvas lifecycle, event handling, and
theming, mirroring the [Leptos integration](/integrations/leptos).

## Add the dependency

```toml title="Cargo.toml"
[dependencies]
rs-grid-core = { path = "../crates/rs-grid-core" }
rs-grid-yew  = { path = "../crates/rs-grid-yew" }
```

## Component API

Yew `Properties` require `PartialEq`, which `GridModel` does not implement, so
the model is passed through a `ModelSlot` wrapper:

```rust
use rs_grid_core::{column::ColumnDef, model::GridModel};
use rs_grid_yew::{GridCanvas, ModelSlot};
use yew::prelude::*;

let columns = vec![
    ColumnDef::new("name", "Name", 200.0),
    ColumnDef::new("email", "Email", 260.0),
];
let data = vec![
    vec!["Alice".into(), "alice@example.com".into()],
    vec!["Bob".into(), "bob@example.com".into()],
];
let model = ModelSlot::new(GridModel::new(columns, data, 32.0, 40.0));

html! {
    <GridCanvas model={model} width="100%" height="600px" />
}
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `model` | `ModelSlot` | required | Wrap a `GridModel` with `ModelSlot::new`. Consumed on first mount. |
| `width` | `AttrValue` | `"100%"` | CSS width, e.g. `"100%"` or `"800px"`. |
| `height` | `AttrValue` | `"600px"` | CSS height, e.g. `"600px"`. |
| `theme` | `Option<Theme>` | optional | Theme; changes apply in place via `set_theme()`. |
| `locale` | `Option<Locale>` | optional | Locale; changes apply in place via `set_locale()`. |
| `on_mount` | `Option<Callback<WebGridCanvas>>` | optional | Called once after mount with a handle. |
| `on_validation_error` | `Option<ValidationErrorCb>` | optional | Fired when a column validator rejects an edit: `(row, col_key, message)`. |

<Tip>
  `GridModel` is consumed (moved) on first mount. For large or infinite
  datasets, build it with `GridModel::with_data_source` instead of an in-memory
  `Vec`. Theming uses the same `--rs-grid-*` CSS variables — see the
  [CSS Variables Reference](/theming/css-variables).
</Tip>

## Full example

```rust title="src/main.rs"
use rs_grid_core::{column::ColumnDef, model::GridModel};
use rs_grid_yew::{GridCanvas, ModelSlot};
use yew::prelude::*;

#[function_component]
fn App() -> Html {
    let model = use_memo((), |_| {
        let columns = vec![
            ColumnDef::new("product", "Product", 220.0),
            ColumnDef::new("price", "Price", 120.0),
        ];
        let data = vec![
            vec!["Keyboard".into(), "49.99".into()],
            vec!["Monitor".into(), "199.00".into()],
        ];
        ModelSlot::new(GridModel::new(columns, data, 32.0, 40.0))
    });

    html! {
        <main style="width: 100vw; height: 100vh;">
            <GridCanvas model={(*model).clone()} width="100%" height="100%" />
        </main>
    }
}

fn main() {
    yew::Renderer::<App>::new().render();
}
```
